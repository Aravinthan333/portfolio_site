## Description

Architected and delivered the full-stack public web platform for Valathi Chit Funds, a government-registered Indian chit fund with branches in Trichy, Madurai, and Chennai.

Built a bilingual (English/Tamil) Next.js application with 70+ React components, a PostgreSQL backend via Prisma, and a REST API consumed by both the public site and an internal admin console. Key deliverables include an online membership application wizard with digital signature capture, a chit fund return calculator with domain-specific financial logic, a bilingual blog CMS with S3 image uploads, and a multi-brand A4 PDF generation pipeline using Puppeteer with Tamil font embedding.

Deployed to AWS EC2 with Docker Postgres, Nginx reverse proxy, PM2 process management, and GitHub Actions CI/CD using Next.js standalone builds.
