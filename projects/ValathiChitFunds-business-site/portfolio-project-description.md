## Short Description (1-2 sentences)

Built the full-stack public website and API for Valathi Chit Funds, a government-registered Indian chit fund. The platform supports bilingual (EN/TA) content, online membership applications with digital signatures, domain-specific return calculators, and a multi-brand PDF pipeline for legal forms - deployed on AWS EC2.

---

## Medium Description (for expanded project cards)

Designed and delivered valathichits.com - a production Next.js platform for a regulated chit fund business operating across three Tamil Nadu branches. The site lets prospective members browse active chit groups, estimate returns, apply online, and download legally compliant application forms. Behind the scenes, a PostgreSQL-backed REST API powers content management for an integrated admin console.

**Stack:** Next.js 16 · React 19 · TypeScript · PostgreSQL · Prisma · AWS (EC2, S3) · Puppeteer · Nginx · GitHub Actions
