## Description

Designed and delivered a production WhatsApp Business operations platform for Valathi Chits, an Indian chit fund company. The internal console enables agents to manage customer conversations, send scheduled bulk template campaigns, track SLA-aware support tickets, and administer the public website - all from a single application.

Built a real-time messaging system with Socket.IO and Redis pub/sub, an async outbound pipeline using BullMQ job queues, and a defensive webhook orchestration layer handling heterogeneous MSG91/Meta Cloud payload formats. Implemented WhatsApp-parity chat features (reply, forward, edit, delete, delivery receipts), a 3-stage campaign scheduler with multi-wave retry, client bulk import from Excel, and role-based auth with email OTP 2FA.

Extended the platform with an integrated website admin module for chit schemes, bilingual blogs, membership application review, and Tamil-language A4 PDF generation via headless Chromium.

Deployed to AWS EC2 via GitHub Actions with Dockerized API, PM2-managed frontend, and Nginx with Let's Encrypt TLS.
