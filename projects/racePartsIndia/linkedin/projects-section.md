## Project 2: Race Parts India - Cloud Infrastructure & CI/CD

**Role:** Freelance DevOps Engineer & Cloud Architect  
**Duration:** [Add your dates]  
**Associated with:** Race Parts India (https://racepartsindia.com)

**Description:**

Provisioned and automated the full production hosting stack for Race Parts India on AWS.

Used Terraform to define EC2 compute, security groups (HTTP/HTTPS public, SSH IP-restricted), SSH key pairs, and least-privilege IAM roles with SSM Session Manager access. Configured Nginx as a high-performance reverse proxy with Let's Encrypt SSL, SPA fallback routing, and immutable static asset caching.

Built reusable GitHub Actions composite workflows for SSH setup, rsync deployment, and remote command execution. Implemented an "images-first" deploy strategy to prevent partial-load UI breakage, plus automated SSL certificate renewal.

Documented the entire setup (Terraform, nginx, GitHub Actions, secrets management) for maintainability and future handoff.

**Skills:** AWS · EC2 · Terraform · Nginx · Let's Encrypt · GitHub Actions · CI/CD · Infrastructure as Code · Linux · rsync · DevOps · Security Groups · IAM
