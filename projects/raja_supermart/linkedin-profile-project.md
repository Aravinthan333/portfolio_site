## Description

Designed and delivered a production grocery e-commerce platform for Raja Supermart, an Indian supermarket. The system includes a customer-facing storefront, an admin operations console, and delivery partner APIs - all backed by a dual-database architecture bridging MongoDB and a legacy Optech SQL Server inventory system.

Built the React + Vite frontend with role-based routing (customer, admin), Redux state management, Recharts analytics dashboards, and a full shopping flow (browse, cart, wishlist, checkout, returns). Developed the Express + MongoDB backend with hierarchical product catalog (Category → Sub-Category → Brand → Variants), JWT cookie authentication, MSG91 OTP verification, AWS S3 image storage, and Redis caching.

Integrated legacy POS stock sync via scheduled cron jobs, implemented phone-order workflows for call-based customers, return order management with delivery partner coordination, and revenue analytics with IST timezone support. Deployed to production at rajasupermart.com with API at api.rajasupermart.com.
